# Use Case: Announcement Cards

Use this for hero banners, release announcements, profile highlights, and promo cards.

## Load These Files

1. `references/announcement-card-template.md`
2. `references/announcement-card-spec.template.json`

## Layout Guidance

- Keep one dominant headline.
- Keep supporting text to 1-3 concise lines.
- Use one focal accent element (badge, CTA, or image) as primary emphasis.
- Maintain high contrast and generous padding.
