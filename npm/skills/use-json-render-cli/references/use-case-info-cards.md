# Use Case: Info Cards

Use this for non-table KPI snapshots, comparisons, and summary cards.

## Select Variant

- KPI overview: `references/kpi-overview-template.md` + `references/kpi-overview-spec.template.json`
- Metric comparison: `references/metric-compare-template.md` + `references/metric-compare-spec.template.json`
- Daily summary: `references/daily-summary-template.md` + `references/daily-summary-spec.template.json`

## Layout Guidance

- Preserve clear hierarchy: title -> key numbers -> supporting context.
- Keep spacing deterministic for repeated snapshots.
- Increase viewport height before reducing type size when content clips.
