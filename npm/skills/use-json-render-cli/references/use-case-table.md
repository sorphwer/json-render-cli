# Use Case: Generic Table

Use this when the user asks for a generic rows/columns table and the schema is not ticket-specific.

## Load These Files

1. `references/compact-table-template.md`
2. `references/compact-table-spec.template.json`

## Layout Guidance

- Use explicit column widths with compact spacing.
- Use `Badge` only for categorical status-like fields.
- Resize the widest content column first if overflow occurs.
- Keep viewport width close to table footprint.
