# Use Case: Flow Summary

Use this for ordered stages, timelines, lifecycle snapshots, or process status summaries.

## Load These Files

1. `references/flow-summary-template.md`
2. `references/flow-summary-spec.template.json`

## Layout Guidance

- Preserve clear stage order (top-to-bottom or left-to-right).
- Keep consistent spacing across stages.
- Separate stages using dividers or bordered containers.
- Set `screenshot.fullPage=true` when stage count is variable.
